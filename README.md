# Radithya Fawwaz Aydin - Portfolio

Welcome to my personal portfolio repository! This website is designed to showcase my journey as a **Data Scientist**, **Machine Learning Engineer**, and **AI Enthusiast**. It features my professional background, technical skills, and a curated list of my data science projects.

## Live Demo
Check out the live version here: **https://radith-portofolio.vercel.app/**

## Tech Stack
This project is built using modern web technologies to ensure performance, responsiveness, and scalability:

* **Core:** [React JS](https://react.dev/) (v18)
* **Build Tool:** [Vite](https://vitejs.dev/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Animations:** [Framer Motion](https://www.framer.com/motion/)
* **Icons:** [Lucide React](https://lucide.dev/) & [Devicon](https://devicon.dev/)
* **Typewriter Effect:** React Simple Typewriter
* **Deployment:** Vercel

## Features
* **Responsive Design:** Fully optimized for Desktop, Tablet, and Mobile devices.
* **Glassmorphism UI:** Modern dark-themed design with blur effects.
* **Interactive Animations:** Smooth scroll reveals and dynamic text typing effects.
* **Data-Driven:** Content is separated from logic (`src/constants/index.js`), making it easy to update.
* **Direct Contact:** Integrated WhatsApp and Email links for fast communication.

## Project Showcase
This portfolio highlights several key projects in Data Science & AI:
1.  **We're Cooked:** Smart recipe recommendation app using Hybrid Filtering (TensorFlow & Scikit-Learn).
2.  **Football Analysis & AI Summary:** Match analysis using IBM Granite LLM via Replicate API.
3.  **Student Depression Prediction:** Binary classification model (89% Recall) using Logistic Regression.
4.  **Movie RecSys (TMDB 5000):** Content-based recommendation system using Cosine Similarity.
5.  **Celebrity Face Recognition:** Image classification using OpenCV and Flask.
6.  **Wine Quality Prediction:** Predictive modeling using XGBoost.
